# Exercise 04 — Nextflow on HPC

**Format:** Instructor demo

This session is a live demonstration of how to run a Nextflow pipeline on an HPC cluster (Vera2/C3SE).

## What will be covered

- Submitting Nextflow jobs via SLURM
- Configuring executors and resource limits in `nextflow.config`
- Using Apptainer (Singularity) containers on the cluster
- Resuming a failed run on HPC
